Volume Reconstruction  1.0
This classes implement a method to reconstruct 3D volumes using 2D ultra sound images. The method used here is a voxel based method that uses the information of the three nearest pixels to compute each voxel value.

Detailed Description

Display VTK images.

This class allows the user to display vtkImages on QT. It can display 2D and 3D scenes
